## 概要
オリジナル曲「二つの世界」のGarageBandプロジェクトのレポジトリです。

作曲・編曲をgithub上で行うとどうなるかという試みです。

各種アレンジなどのプルリクエスト、ウェルカムです。<br />
forkして歌ってみたとかも面白いかも。
